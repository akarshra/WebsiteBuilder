🚀 GenWeb.ai

AI-powered website builder that generates modern, responsive websites using AI.

Users describe their idea, and the AI generates production-ready HTML/CSS websites instantly.

📦 Installation
Clone Repository
git clone https://github.com/your-username/genwebai.git
cd genwebai
⚙️ Backend Setup
cd server
npm install

Create .env file:

PORT=8000
MONGODB_URL=your_mongodb_connection_string
JWT_SECRET=your_secret_key
OPENROUTER_API_KEY=your_openrouter_key
FRONTEND_URL=http://localhost:5173
STRIPE_SECRET_KEY=your_stripe_secret
STRIPE_WEBHOOK_SECRET=your_webhook_secret

Run backend:

npm run dev
🎨 Frontend Setup
cd client
npm install
npm run dev
